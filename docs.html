AI Server API Documentation

Unified API for GitHub Models, Groq, and Google Gemini

Base URL: https://rebyb-llms.onrender.com

Overview

This server provides a simple REST API to interact with multiple AI providers:

  • ✅ GitHub Models (OpenAI-compatible)
  • ✅ Groq
  • ✅ Google Gemini

Common Request Parameters

{
  "prompt": "Your question or input text",
  "systemPrompt": "Optional system instruction",
  "model": "Optional model name",
  "apiKey": "Optional API key override"
}

1️⃣ Unified AI Endpoint (Auto Fallback)

POST / GET /ai 
curl -X POST https://rebyb-llms.onrender.com/ai \
-H "Content-Type: application/json" \
-d '{
  "prompt": "Explain JavaScript closures",
  "systemPrompt": "You are a helpful teacher"
}'
  
{
  "ans": "Explanation text...",
  "errors": []
}

2️⃣ GitHub Models API

POST / GET /ai/github-model 
fetch("https://rebyb-llms.onrender.com/ai/github-model", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    prompt: "Write a Node.js example",
    systemPrompt: "You are a senior developer"
  })
});

3️⃣ Groq API

POST / GET /ai/groq 
curl "https://rebyb-llms.onrender.com/ai/groq?prompt=Explain REST APIs"

4️⃣ Gemini API

POST / GET /ai/gemini 
fetch("https://rebyb-llms.onrender.com/ai/gemini", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    prompt: "Summarize HTTP status codes"
  })
});

Environment Variables

GITHUB_TOKEN=your_github_models_token
GROQ_API_KEY=your_groq_api_key
GEMINI_API_KEY=your_gemini_api_key
